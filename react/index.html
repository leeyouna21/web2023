React

React

React는 Facebook에서 개발한 UI 라이브러리입니다.

React는 사용자 인터페이스를 구축하기 위한 선언적이고 효율적인 방법을 제공합니다. React는 컴포넌트 기반 아키텍처를 사용하여 사용자 인터페이스를 작성하는 것을 간편하고 재사용 가능하게 만들어 줍니다.

REACT 설치

Node.js 설치
https://nodejs.org/ko

React를 사용하려면 먼저 Node.js와 npm이 설치되어 있어야 합니다. Node.js는 자바스크립트 런타임입니다. npm은 Node.js 패키지 매니저로, 다양한 패키지를 다운로드하고 설치할 수 있습니다.

npx create-react-app react1

REACT 기본 개념 익히기

hello world 출력하기

import React from "react";
import  ReactDOM  from "react-dom/client";
    
// const root = ReactDOM.(document.getElementById('root'));
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<h1>Hello world</h1>);

1. JSX

JSX는 JavaScript XML의 약자로, React에서 UI를 작성하기 위한 문법입니다. JSX는 HTML과 비슷한 형태의 코드를 작성하여 UI를 표현할 수 있도록 해주는데, JavaScript 코드와 함께 혼합하여 사용할 수 있습니다. 

const name = "leeyouna";
const hello = <h1>hello {name}</h1>;


const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);

2. JSX

객체와 함수를 같이 사용한 경우 

function helloName (){
    return name.nick;
}

const name = {
    nick : "leeyuna"
}

const hello = <h1>hello, {helloName()}</h1>

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);
