01.선언적 함수

선언적 함수(declarative function)는 자바스크립트에서 함수를 정의하는 방법 중 하나입니다.
{
    function func () {
        document.write("실행되었습니다.
");
}
    func();
}
결과 확인하기
실행되었습니다.
함수를 선언할 때 function 키워드를 사용하여 정의하는 방법입니다. 함수 선언문 이라고도합니다.
선언적 함수는 함수를 정의하는 시점에서 함수 객체가 생성되며, 호이스팅(hoisting)이 가능합니다.
호이스팅이 가능하다는 것은 함수가 정의되기 이전에 함수를 호출할 수 있다는 것을 의미합니다.
따라서, 함수를 선언문으로 정의하면 코드의 가독성과 유지보수성을 높일 수 있습니다.

02. 익명 함수

변수는 데이터를 저장하는 "저장소" 이며 이 저장소에는 숫자, 문자, 함수, 객체 등을 저장할수 있습니다.
{
    const func = function() {
        document.write("실행되었습니다.
");
}
    func();
}
결과 확인하기
실행되었습니다.
익명 함수는 함수 표현식(function expression)이나 콜백 함수(callback function) 등에서 주로 사용됩니다.
함수를 변수에 할당하거나, 객체의 속성 값으로 함수를 할당할 때 사용됩니다.

03. 매개변수 함수

매개변수 함수(parameter function)는 자바스크립트에서 함수의 인자로 전달되는 함수를 말합니다 
{
    function func(str) {
        document.write(str);
}
    func("실행되었습니다.");
}
결과 확인하기
실행되었습니다.
매개변수 함수를 이용하면 함수의 재사용성과 확장성을 높일 수 있습니다.
함수의 로직 중 일부를 매개변수 함수로 분리하여, 함수 호출 시에 동적으로 로직을 변경하거나 확장할 수 있기 때문입니다.

04. 리턴값 함수

리턴값 함수(return function)는 자바스크립트에서 함수가 실행된 결과를 반환하는 함수를 말합니다.
{
    function func() {
        const str = "실행되었습니다.";
        return str;
}
    document.write(func());
}
결과 확인하기
실행되었습니다.
함수 내부에서 연산, 조작, 처리한 결과를 반환하여, 이를 다른 변수나 함수에서 활용할 수 있습니다.

05. 화살표함수: 선언적함수

화살표 선언적 함수(arrow function declaration)는 자바스크립트에서 함수를 정의하는 방법 중 하나입니다.
{
    func = () => {
        document.write("실행되었습니다.
");
}
    func ();
}
결과 확인하기
실행되었습니다.
일반적인 함수 선언문과 달리, 화살표 함수를 이용하여 간결하고 명확한 함수를 작성할 수 있습니다. 화살표 함수는 함수의 인자와 로직을 괄호로 묶어 화살표 표기법(=>)으로 구성됩니다.
화살표 함수는 일반적으로 함수 표현식(function expression)으로 사용되며,
변수에 할당되어 다른 함수의 인자로 전달되거나 콜백 함수(callback function)로 사용됩니다.

화살표 함수를 이용하여 간결하고 명확한 함수를 작성할 수 있습니다.
화살표 익명 함수는 함수의 인자와 로직을 괄호로 묶어 화살표 표기법(=>)으로 구성됩니다.
화살표 함수는 일반적으로 함수 표현식(function expression)으로 사용되며,

08. 화살표 함수 : 리턴값 함수

화살표 리턴값 함수(arrow function with return value)는 자바스크립트에서 화살표 함수를 작성할 때, 함수의 반환 값을 지정하는 방법을 말합니다.
{
    func = () => {
        const str = "실행되었습니다.
";
        return str;
}
    document.write(func());
}
결과 확인하기
실행되었습니다.
화살표 함수는 함수 본문 내부의 문장이 하나일 경우에는 중괄호와 return 키워드를 생략할 수 있으며,
이때는 해당 문장의 결과값이 함수의 반환 값으로 자동 지정됩니다.

09. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값

화살표 함수 익명 함수와 매개변수 리턴함수를 사용한 예제입니다
{
    a = () => {
        document.write("실행1");
}
    a();
    const b = () => {
        document.write("실행2");
}
    b();
    const c = (str1, str2, str3,) => {
        document.write(str1, str2, str3,); //선언적 함수를 화살표 함수로 바꾸는 것은 함수처럼 안보이기도하며 가독성이 떨어지기에 익명함수로 바꿔서 많이 사용함
}
    c("실행3","실행4","실행5");
    d = (x, y, z) => {
        return x+y+z;
}
    document.write(d(1,2,3));
}
결과 확인하기
실행1
실행2
실행3
실행4
실행56
위 순대로 익명 화살표 함수 a를 선언하고, 이를 호출하여 "실행1"을 출력합니다.
상수(const) b에 익명 화살표 함수를 할당하고, 이를 호출하여 "실행2"를 출력합니다.
세 개의 문자열 매개변수를 받는 익명 화살표 함수 c를 선언하고, 이를 호출하여 "실행3실행4실행5"를 출력합니다.
매개변수와 인자는 쉼표로 구분되며, 여기서는 인자를 쉼표 없이 나열하였습니다.
세 개의 숫자 매개변수를 받는 익명 화살표 함수 d를 선언하고, 이를 호출하여 1, 2, 3을 더한 결과인 6을 출력합니다.
함수 내에서 return 키워드를 사용하여 값을 반환합니다.

13. 함수 유형 : 함수와 매개변수를 이용한 형태

함수는 입력값을 받아 처리를 하여 출력값을 반환하는 일련의 과정을 수행하는 코드 블록입니다. 함수의 입력값은 매개변수(parameter)를 통해 전달되며, 함수가 반환하는 값은 return문을 통해 지정됩니다.

{
    function func(num, str1, str2){
        document.write(num + ". " + str1 +"가 " +str2 + "됐습니다.
");
}
        func("1", "함수","실행");
        func("2", "자바스크립트","실행");
        func("3", "리액트","실행");
}
결과 확인하기
1. 함수가 실행됐습니다. 2. 자바스크립트가 실행됐습니다. 3. 리액트가 실행됐습니다.
함수와 매개변수를 이용한 함수는 입력값에 따라 출력값이 달라질 수 있는 유연한 코드를 작성할 수 있게 해줍니다. 이를 이용하여 프로그래밍 작업을 더욱 효율적으로 수행할 수 있습니다.

19. 함수 유형 : 객체 생성자 함수

객체 생성자 함수란, 객체를 생성하기 위한 함수를 말합니다. 객체 생성자 함수를 이용하여 객체를 생성하면, 객체를 초기화하고 필요한 속성을 설정할 수 있습니다.

{
    function Func(num, name, word){
        this.num = num;
        this.name = name;
        this.word = word;
        
        this.result = function(){
            document.write(this.num + ". " + this.name +"가 " +this.word + "됐습니다.
");
        }
}

    //인스턴스 생성
    const info1 = new Func(1, "함수", "실행");
    const info2 = new Func(2, "자바스크립트", "실행");
    const info3 = new Func(3, "리액트", "실행");

    info1.result();
    info2.result();
    info3.result();
}
결과 확인하기
1. 함수가 실행됐습니다. 2. 자바스크립트가 실행됐습니다. 3. 리액트가 실행됐습니다.
객체 생성자 함수를 이용하면, 객체를 생성하고 초기화하는 코드를 함수로 구성할 수 있으며, 객체를 생성하는 코드를 단순화할 수 있습니다.

20. 함수 유형 : 프로토타입 함수

프로토타입 함수(Prototype function)는, 함수를 정의하기 전에 함수의 형태를 먼저 정의해둔 뒤, 나중에 함수의 구현을 추가하는 방식입니다. 

{
    function Func(num,name, word){
        this.num = num;
        this.name = name;
        this.word = word;
}

    Func.prototype.result = function(){
        document.write(this.num + ". " + this.name +"가 " +this.word + "됐습니다.
");
}
    const info1 = new Func(1, "함수", "실행");
    const info2 = new Func(2, "자바스크립트", "실행");
    const info3 = new Func(3, "리액트", "실행");

    info1.result();
    info2.result();
    info3.result();
}
결과 확인하기
1. 함수가 실행됐습니다. 2. 자바스크립트가 실행됐습니다. 3. 리액트가 실행됐습니다.
